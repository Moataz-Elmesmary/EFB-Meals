# Meals Fleet — Food Request System

A lightweight full-stack scaffold for a kitchen request system (Arabic/English) inspired by a fleet project. Includes:

- Backend: Express + SQLite, file uploads, email automation (nodemailer), SAP integration stub (writes to `SalesOrder`).
- Frontend: Vite + React minimal app with i18n toggle and animated badges (placeholder).

See `backend/README.md` and `frontend/README.md` for start instructions.
